The Pre Function 

   Here I am using the pre function to inform the code that the text in this section is pre-formatted. What 
this means is that the indents, line breaks, and font which I am typing the code with will in fact be kept
and so the html document text will look identical to the text that I wrote when coding it. Notice how it does
look different to the text above? That is because I have added line breaks based on VSCode's display size,
and the font is the same font as that which is used in VSCode. Neat, huh?
